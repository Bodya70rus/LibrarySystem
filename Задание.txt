Создайте консольное приложение, которое моделирует систему управления библиотекой. В этом приложении должны быть реализованы следующие классы:

Класс Книга, содержащий следующие поля:

Название книги
Автор
Год издания

Класс Библиотека, содержащий следующие поля:
Массив объектов класса Книга, представляющий каталог книг в библиотеке

Класс МенеджерБиблиотеки, содержащий следующие методы:
Добавление книги в каталог библиотеки
Поиск книги по названию или автору
Вывод списка всех книг в каталоге библиотеки

Главный класс Main, содержащий метод main, инициализирующий экземпляр класса МенеджерБиблиотеки и позволяющий пользователю взаимодействовать с системой управления библиотекой через команды в консоли.

Вот некоторые варианты команд, которые можно реализовать:

"добавить": добавить новую книгу в каталог библиотеки
"поиск по названию": поиск книг по названию и вывод результатов
"поиск по автору": поиск книг по автору и вывод результатов
"список": вывод всех книг в каталоге библиотеки
Вы можете реализовать дополнительные команды и функции по вашему усмотрению.

Успешное выполнение этого задания позволит вам применить основные принципы ООП, такие как инкапсуляция, наследование и полиморфизм, и применить их на практике при работе с Java. Удачи!
